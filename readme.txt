
# Descrição do Proejeto

API Desenvolvida para fazer o controle de estoque de materiais e medicamentos.
Projeto desenvolvido com base em conhecimentos adquiridos na Semana Omnistack 11, da RocketSeat


## Tecnologias utilizadas

Projeto desenvolvido em JavaScript, com NodeJS e Express, para o banco de dados foi utilizado 
o ORM Sequelize em conjunto com um banco de dados Postgres.

### Motivação para desenvolvimento do projeto

O projeto foi desenvolvido buscando atender a necessidade de controlar o estoque de materiais 
e medicamentos do local onde trabalho, foi feito baseado em movimentação de itens, visto que
há vários itens que são do mesmo lote e precisam ser agrupados, e também, para deixar o fluxo de trabalho
parecido com o que ocorre no serviço.
